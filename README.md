📅 Smart Scheduling System

A web-based task scheduling system that optimizes task selection using the Weighted Interval Scheduling Algorithm (Greedy + Dynamic Programming). The system helps users manage tasks, avoid scheduling conflicts, and maximize overall profit or priority through intelligent schedule generation.

🌟 Features

📝 Task Management
• Add, edit, and delete tasks
• Define task name, start time, end time, and priority/profit
• Store tasks securely using SQLite database

📊 Dashboard & Analytics
• View total number of tasks
• Track total profit across all tasks
• Monitor recently added tasks

🧠 Smart Schedule Generation
• Uses Greedy sorting and Dynamic Programming
• Selects optimal non-overlapping tasks
• Maximizes total profit/priority
• Automatically removes scheduling conflicts

📈 Schedule Visualization
• Displays all tasks sorted by completion time
• Shows optimal task selection results
• Generates a conflict-free timeline view
• Highlights total achievable profit

💾 Database Integration
• SQLite database for persistent storage
• Automatic task management and retrieval
• Efficient data handling through Flask backend

🎨 User Interface
• Responsive Bootstrap-based design
• Interactive dashboard and task management pages
• Clean and modern scheduling interface

🛠️ Technologies Used

• HTML5
• CSS3
• Bootstrap 5
• Dynamic Programming
• Greedy Algorithms

🎯 Key Highlights
